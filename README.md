# Online Book Store

## 📚 Overview

Online Book Store is a **Java-based web application** that allows users to browse and purchase books online.
The project demonstrates a typical **Java web application with Maven build automation and CI/CD using Jenkins**.

---

## 🚀 Features

* Browse available books
* View book details
* Purchase books online
* Simple and clean UI
* Maven-based build system
* CI/CD integration using Jenkins

---

## 🛠️ Tech Stack

| Technology     | Purpose                |

## ⚙️ Build the Project

Clone the repository:

```
git clone https://github.com/Kavyagundeti/onlinebookstore.git
```

Navigate to project folder:

```
cd onlinebookstore
```

Build using Maven:

```
mvn clean package
```

Generated WAR file:

```
target/onlinebookstore.war
```

---

## 🔄 CI/CD Pipeline

This project uses **Jenkins** for automated builds.

Pipeline steps:

1. Pull code from GitHub
2. Build using Maven
3. Generate WAR file
4. Deploy to server
